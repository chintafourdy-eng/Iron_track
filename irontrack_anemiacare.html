🩺 Skrining Risiko Anemia

Jawab pertanyaan berikut dengan jujur untuk mengetahui risiko anemia kamu.

A. Gejala Anemia

Jawab Ya atau Tidak untuk setiap pernyataan berikut.

1. Kelopak mata bawah saya pucat
2. Lidah saya pucat
3. Kuku tangan dan kaki saya pucat
4. Kuku tangan dan kaki saya mudah rusak
5. Saya sering merasa pusing dan kunang-kunang
6. Saya sering merasa letih, lemah, lesu, lunglai, dan lalai (5L)
7. Ada anggota keluarga saya yang menderita anemia
8. Saya pernah transfusi darah atau tambah darah
9. Saya sering tidak nafsu makan
10. Saya sering makan sayur dan buah
11. Saya memiliki kebiasaan sarapan
12. Saya memiliki riwayat penyakit dalam 1 bulan terakhir
B. Riwayat Menstruasi (Khusus Perempuan)

Lewati bagian ini jika kamu laki-laki.

C. Pola Makan & Nutrisi

Jawab Ya atau Tidak untuk setiap pernyataan.

1. Saya rutin makan 3 kali sehari
2. Saya mengonsumsi makanan sumber zat besi (daging, kacang-kacangan, sayuran hijau)
3. Saya mengonsumsi buah mengandung vitamin C (jambu biji, jeruk, mangga)
4. Saya sering minum teh/kopi setelah makan
